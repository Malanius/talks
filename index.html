🚀 Getting Started

To run the presentations:

  1. Install dependencies: npm install
  2. Start the server: npm start
  3. Open http://localhost:3000 in your browser

📝 Adding New Presentations

To add a new presentation:

  1. Create a new directory with your presentation name
  2. Add a README.md file with your markdown content
  3. Use --- for horizontal slides and `-` or --- for vertical slides
  4. Access it via /your-presentation-name
